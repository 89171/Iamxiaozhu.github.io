sublime Text SFTP使用密钥远程登录Linux 上传文件

设置host,username和remote_path就不多讲了，需要注意的是设置ssh_key_file,因为sublime的SFTP使用的是PSFTP，需要使用ppk后缀的秘钥，我们可以使用puttygen进行转换。

puttygen下载地址

使用方法 ：
打开puttgen>点击Load>弹出的文件选择器切换类型为All File>选择秘钥>File>Save private key 即可

sublime中文件夹右键SFTP/FTP ，就可以上传，下载，同步文件夹

Preferences>Package Settings>SFTP>Key Bindings - User，在打开的文件数组中添加以下对象，直接按ctrl+u即可上传文件，自己也可以配置其他快捷键

{"keys":["ctrl+u"],"command":"sftp_upload_file"}

sublime Text 3 emmet自定义HTML模板

1.下载Emmet

https://github.com/sergeche/emmet-sublime

2.打开sublime Package程序包目录。windows一般在C:\Users\userName\AppData\Roaming\Sublime Text 3\Packages\下。在Package目录中创建Emmet目录,并将下载的文件解压复制到此目录

3.Emmet/emmet/snippets.json中可以配置HTML模板内容.

a)找到html.abbreviations.doc即可定义HTML中header和body标签中存放内容.

b)html.abbreviations.doc字符串中可以应用变量,变量定义在variables中,字符串中使用使用${变量名}即可.

4.定义后重启sublime,创建HTML文件,输入!或html:5,按下tab键即可.
